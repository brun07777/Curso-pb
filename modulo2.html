1. Fatorial

O código abaixo realiza o cálculo do fatorial de um número inserido pelo usuário. Primeiro, ele solicita que o usuário insira um número, que é armazenado na variável n. Em seguida, o valor de n é copiado para a variável a. A variável b é inicializada como 1 e será usada para acumular o resultado do fatorial. O loop for é então executado, iterando de a até 2 (exclusivo) de forma decrescente. Em cada iteração, o valor de b é multiplicado pelo valor atual de a, acumulando assim o produto dos números de n até 2. Finalmente, o código imprime o valor de b, que é o fatorial do número fornecido pelo usuário.


    n = int(input("\nEntre com um numero para calcular seu fatorial: "))
    a=n
    b=1
    for a in range (a,1,-1):
    b *= a
    print(f"\nO resultado eh = {b}")

2. Fibonacci

Este programa gera a sequência de Fibonacci para uma quantidade de termos especificada pelo usuário. Ele começa pedindo um número n, que define quantos termos da sequência serão calculados. Os dois primeiros termos da sequência, 0 e 1, são definidos inicialmente e impressos. A seguir, o programa calcula os próximos termos usando um laço for, onde cada novo termo é a soma dos dois anteriores. Esses valores são impressos na sequência, gerando a sequência de Fibonacci até o termo n.


    n = int(input("Entre com um número"))
    a=0
    b=1
    print(f"sequencia de Fibonacci:{a} {b}", end = ' ')
    x = n-2
    for i in range(1,x+1):
        c=a+b
        print(c, end =' ')
        a=b
        b=c

3. Soldados

O programa abaixo realiza a fila de soldados em que a cada fila mais um soldado é adicionada, a 1ª fila tem 1 soldado a 2ª tem 2 soldados e assim por diante. O programa permite que o usuário entre com a quantidade de soldados, e usa o while para formar as filas. No final imprimimos o número de soldados, número de filas e soldados na última fila


    fila=1
    n = int(input("\nEntre com o numero de soldados:"))
    sol=n
    while sol > fila:
    sol -= fila
    fila += 1
    print(f"\nO numero de soldados {n}, o numero de filas {fila},o numero de soldados na ultima fila {sol}")

4. Primos

Este programa descobre se o número inserido pelo usuário é primo ou não. Ele descobre isso por meio do módulo dos números em um laço até chegar no número inserido pelo usuário. Se o número tiver módulo 0 apenas 2 vezes quer dizer que o número é primo, mas se ele tiver mais de 2 números ele não é primo.


    num = int(input("Entre com um valor:"))
    cont = 0
    for i in range (1,num+1):
        if num%i == 0:
            cont = cont + 1

    if cont == 2:
        print("Eh primo")
    else:
        print("Nao eh primo")

5. Vetores

Este é um program simples que declara um vetor de acordo com o número inserido pelo usuário. O usuário também deve colocar o valor em cada posição do vetor para preenchê-lo


    n = int(input("Entre com um numero = "))
    vetor=[]
    for i in range(0, n+1):
        vetor.append(int(input(f"Entre com a posiçao{i} = ")))
    for i in range(0,n+1):
        print(vetor[i])

6. Matrizes

Este é um programa que declara uma matriz de acordo com o que o usuário inserir nas variáveis 'linhas' e 'colunas'. E depois o usuário decide o valor das posições da matriz declarada.

import numpy as np
    linhas = int(input("Digite o número de linhas: "))
    colunas = int(input("Digite o número de colunas: "))
    matriz = np.zeros((linhas, colunas))
    for i in range(linhas):
        for j in range(colunas):
            valor = int(input(f"Digite o valor para a posição [{i}][{j}]: "))
            matriz[i, j] = valor
    for i in range(linhas):
        print()
        for j in range(colunas):
            print(f"[{matriz[i,j]}]",end=' ')

7. Criação de Arquivos

Neste código o usuário pode criar um arquivo. Ele começa inserindo o nome do arquivo sem a extensão dele pois mais tarde concatenaremos o valor com o .txt o arquivo é criado tipo 'w' para o usuário poder escrever nele. Na linha seguinte o usuário escreve o que quiser dentro do arquivo e depois não podemos nos esquecer de fechá-lo


    nome = input("Escreva o nome do arquivo:")
    arquivo = open(nome+".txt", "w")
    valor = input("Escreva o que você quiser:")
    arquivo.write(valor)
    arquivo.close()

8. Leitura de Arquivos

Este código é muito parecido com o anterior mas a diferença é que ele utiliza a função 'r' que serve apenas para leitura. E utiliza a função arquivo.read() para ler o que está no arquivo


    nome = input("Escreva o nome do arquivo:")
    arquivo = open(nome+".txt", "r")
    conteudo = arquivo.read()
    print(conteudo)
    arquivo.close()

9. Exclusão de Arquivos

Este código apesar de parecer diferente é muito parecido com os anteriores mas dessa vez é utilizada a biblioteca 'os' que será responsável por deletar o arquivo. Usamos um if para verificar se o arquivo existe e se não existir ele apresenta mensagem de erro.


    import os
    nome = input("Digite o nome do arquivo que deseja excluir: ")
    if os.path.exists(nome+".txt"):
        os.remove(nome+".txt")
        print(f"O arquivo '{nome+".txt"}' foi excluído com sucesso.")
    else:
        print(f"O arquivo '{nome+".txt"}' não foi encontrado.")
